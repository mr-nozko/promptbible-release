# Prompt Bible - Release Repository

This repository contains release installers for Prompt Bible, a lightweight desktop utility for capturing, organizing, and searching prompts.

## About This Repository

This is a **release-only repository**. It contains:
- Compiled installers for Windows, macOS, and Linux
- Release notes and changelogs
- Download links for each platform

**Source code is not included in this repository.** For the source code, please visit the main repository.

## Downloads

Visit the [Releases](https://github.com/mr-nozko/promptbible-release/releases) page to download the latest version for your platform.

### Supported Platforms

- **Windows**: `.exe` (NSIS installer) and `.msi` (MSI installer)
- **macOS**: `.dmg` (Disk Image)
- **Linux**: `.AppImage` and `.deb` (Debian package)

## Installation

### Windows
1. Download the `.exe` or `.msi` installer
2. Run the installer and follow the setup wizard
3. Launch Prompt Bible from the Start menu

### macOS
1. Download the `.dmg` file
2. Open the DMG and drag Prompt Bible to Applications
3. Launch from Applications

### Linux
- **AppImage**: Make executable (`chmod +x`) and run
- **Debian**: Install with `sudo dpkg -i prompt-bible_*.deb`
